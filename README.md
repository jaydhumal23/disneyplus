# Disney+ Clone

A responsive, pixel-perfect Disney+ inspired movie streaming interface built using **React (Vite)** and **Tailwind CSS**. This project fetches real-time movie and TV show data using **The Movie Database (TMDB) API**.

It is a purely client-side application deployed on **Vercel**.

#### <a href="https://disneyplus-1jay.vercel.app/" traget="_blank">Live</a>

## Tech Stack

- **Frontend Framework:** React.js (initialized via Vite for speed)
- **Styling:** Tailwind CSS
- **Data Source:** TMDB API (The Movie Database)
- **Deployment:** Vercel
- **HTTP Client:** Axios

---

## Key Features

- **Responsive Design:** Fully responsive layout that works seamlessly on Desktop, Tablet, and Mobile devices.
- **Real-time Data:** Fetches trending movies, top-rated shows, and genre-specific content dynamically from TMDB.
- **Client-Side Architecture:** No backend server required; all data fetching logic is handled directly within the React client components.
